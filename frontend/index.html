Sobre a Aplicação
Análise de Dados de Sensoriamento Remoto

Esta aplicação utiliza o Google Earth Engine para análise de dados de sensoriamento remoto, permitindo o processamento e visualização de informações geoespaciais em tempo real.

Funcionalidades Principais
  • Análise de Índices de Vegetação: NDVI e EVI para monitoramento da vegetação
  • Visualização RGB: Composições coloridas para análise visual
  • Dados de Precipitação: Análise temporal com dados CHIRPS
  • Múltiplas ROIs: Gerenciamento de várias regiões de interesse
  • Exportação de Dados: Download em formato CSV
  • Download de Imagens: Arquivos TIFF para análise offline
Fontes de Dados
  • Sentinel-2: Imagens ópticas de alta resolução (10-20m)
  • CHIRPS: Dados de precipitação com resolução de 5.5km
  • Google Earth Engine: Plataforma de processamento em nuvem
